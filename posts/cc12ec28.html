在centos上搭建git服务器并自动同步代码

本文最后更新于：14 天前

本篇内容用来讲述如何将 hexo 博客部署到云服务器上。
只要通过三步即可成功部署：

  1. 云服务器端 git 的配置
  2. Nginx 的配置
  3. 本地端 hexo 的设置更改

云服务器端配置 git

可以直接使用 yum install git 安装，也可以使用编译安装，如下：

    • 到官网找一个新版稳定的源码包下载到 /usr/local/src 文件夹里

    • 进入 git-2.16.2.tar.gz 目录下

    • 执行编译

    • 安装 git 到 /usr/local/git 目录下
